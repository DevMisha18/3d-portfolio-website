About My Portfolio

This portfolio showcases three projects that demonstrate my skills in web development:

  • Asteroid Game: A simple yet fun game built with JavaScript.
  • Movie App: A dynamic React + Tailwind app powered by API fetch.
  • Finance Tracker: My most complex project, featuring React, Tailwind, and Firebase for authentication and database functionality.

A full-stack finance tracking app built with React, Tailwind CSS, and Firebase. It features authentication (sign-in/up with email and password) and a real-time database for storing income and expenses.
